# OWASP Juice Shop VAPT

## Project Overview

This project demonstrates a Web Application and API Vulnerability Assessment (VAPT) conducted against OWASP Juice Shop using Burp Suite Community Edition.

The assessment focused on identifying vulnerabilities related to:

- Authentication Security
- Authorization Security
- JWT Security
- API Security
- User Enumeration
- Input Validation
- Access Control

---

## Project Objective

The objective of this project was to perform a structured security assessment of the OWASP Juice Shop web application by identifying and validating vulnerabilities related to authentication, authorization, API security, JWT security, input validation, user enumeration, and access control.

---

## Key Findings

| Vulnerability | Severity |
|--------------|----------|
| SQL Injection Authentication Bypass | Critical |
| Broken Access Control (BOLA) | High |
| JWT Sensitive Information Disclosure | Medium |
| Reflected Cross-Site Scripting (XSS) | Medium |
